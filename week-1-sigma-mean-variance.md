# Sigma Notation Mean and Variance
Set of numbers: X = {X₁ ... Xₙ}
Then the mean is:
		  n
μX = 1/n( ∑ X:)
		  i=1

and the variance is:
		  n
σ² = 1/n [∑ (Xᵢ - μX)²]
 ⁿ 		  i=1

σX = √σ² = Standard Deviation

Z = {1, 5, 12}
|Z| = 3 (cardinality)
μZ = (1 + 5 + 12) / 3 = 6
